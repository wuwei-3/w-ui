# 介绍

BI-UI 是一款基于 [Vue.js 3+](https://cn.vuejs.org/) 的前端 UI 组件库，用于脑科学组-国家项目条线--前端页面快速开发

## 特性

- 基于 Vue 开发的 UI 组件
- 使用 npm + babel 的工作流，支持 ES2015
- 提供友好的 API，利用 Element UI 组件库，二次封装组件。
