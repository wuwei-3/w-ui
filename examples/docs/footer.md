## 页脚

通用页脚，用于标注系统版权。
::: demo

```html
<b-footer bgColor="#ccc" color="white"
  >Copyright © iflytek, All Rights Reserved.</b-footer
>
```

:::

### Attributes

| 参数    | 说明     | 类型   | 可选值 | 默认值  |
| ------- | -------- | ------ | ------ | ------- |
| color   | 字体颜色 | String | -      | #d9d9d9 |
| height  | 高度     | Number | -      | 40      |
| bgColor | 背景颜色 | String | -      | #f7f8f8 |
