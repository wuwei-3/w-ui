## Button 按钮

```html
  <b-button :weight="'200px'" :height="'40px'" :bgColor="'#0070f5'">
      <b-font :color="'#fff'" :size="'f-medium'">按钮</b-font>
  </b-button>
```

  <b-button :weight="'200px'" :height="'40px'" :bgColor="'#0070f5'">
    <b-font :color="'#fff'" :size="'f-medium'">按钮</b-font>
  </b-button>


### Attributes

| 参数    | 说明         | 类型   | 可选值 | 默认值  |
| ------- | ------------ | ------ | ------ | ------- |
| weight  | 宽度         | String | -      | 200px   |
| height  | 高度         | String | -      | 40px    |
| color   | 字体颜色     | String | -      | #fff    |
| bgColor | 背景色       | String | -      | #0070f5 |
| icon    | 图标--不可用 | String | -      | —       |
